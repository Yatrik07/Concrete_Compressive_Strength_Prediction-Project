Warning

If you are trying to ReTrain on the Heroku server :
https://ml-concrete-strength.herokuapp.com/

Then it won't work because:
-> Heroku is a free server and it gives only 200mb RAM , but Model Retraining would take more than 200 mb RAM so in that case server may crash.

If want to check "ReTrain" then Try this steps:

You can check it on your own system :
-> Open command prompt
-> cd Project_Directory # Change to the directory you want to clone project
-> git clone https://github.com/Yatrik07/Concrete_Compressive_Strength_Prediction-Project.git
-> cd Concrete_Compressive_Strength_Prediction-Project
-> pip install -r requirements.txt
-> python app.py
-> Click on ReTrain.
